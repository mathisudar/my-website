8.Array Methods

JavaScript Arrays

The concat() Method

The concat() method merges (concatenates) arrays:

Example Code


                    
                <script>

                const array1 = ["Cecilie", "Lone"];
                const array2 = ["Emil", "Tobias", "Linus"];
                const array3 = ["Robin", "Morgan"];
                
                const myChildren = array1.concat(array2, array3);
                
                document.getElementById("demo8a").innerHTML = myChildren;
                
                </script>

JavaScript Arrays

The join() Method

The join() method joins array elements into a string.

It this example we have used " * " as a separator between the elements:

Example Code


                            
                        <script>
        
                        const fruits = ["Banana", "Orange", "Apple", "Mango"];
                        document.getElementById("demo8b").innerHTML = fruits.join(" * ");
                        
                        </script>
