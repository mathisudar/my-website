1.What can JS Do ?

JS can change HTML content.

JavaScript can change the style of an HTML element.

JavaScript can hide HTML elements.

Example Code


        
        <button type="button" onclick="document.getElementById('demo1').
                        innerHTML = 'Hello Java Script!'">Change Content!
        </button>
            
        <button type="button" onclick="document.getElementById('demo2').
                        style.fontSize='35px'">Change Style!
        </button>
        
        <button type="button" onclick="document.getElementById('demo3').
                        style.display='none'">Hide Content!
        </button>
