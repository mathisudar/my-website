7.Arrays

JavaScript Arrays

Looping an Array

Example Code


                    
                <script>

                const fruits = ["Banana", "Orange", "Apple", "Mango"];
                let fLen = fruits.length;            
                let text = "<ul>";

                for (let i = 0; i < fLen; i++) { 
                    text +="<li>" + fruits[i] + "</li>" ; 
                }                 
                text +="</ul>" ;

                document.getElementById("demo7a").innerHTML=text;        
                
                </script> 
                
                
                More Info:

                <li>must always be inside a <ul> (or<ol>)

                <ul>
                <li>Apple</li>
                <li>Banana</li>
                <li>Orange</li>
                </ul>
