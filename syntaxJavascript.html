JavaScript Statements

A JavaScript program is a list of statements to be executed by a computer.

Example Code


                    
                <script>
                var x, y, z; // Declare 3 variables
                x = 5; // Assign the value 5 to x
                y = 6; // Assign the value 6 to y
                z = x + y; // Assign the sum of x and y to z
                
                document.getElementById("demo4a").innerHTML =
                "The value of z is " + z + ".";
                </script>

JavaScript Numbers

Number can be written with or without decimals.

Example Code


                                
                <script>
                document.getElementById("demo4b").innerHTML = 10.50;
                </script>

JavaScript Strings

Strings can be written with double or single quotes.

JavaScript is Case Sensitive

Example Code


                                        
                <script>
                document.getElementById("demo4c").innerHTML = 'John Doe';

JavaScript Variables

In this example, x is defined as a variable. Then, x is assigned the value of 6:

Example Code

                                                
                <script>
                var myValue;
                myValue = 6;
                
                document.getElementById("demo4d").innerHTML = "The value of x is " + myValue;
                </script>

JavaScript Operators

JavaScript uses arithmetic operators to compute values (just like algebra).

Example Code

                                                
                <script>
                document.getElementById("demo4e").innerHTML = 5 * 10;

The Assignment (=) Operator

The = operator is used to assign values to variables.

Example Code

                                                
                <script>
                var x = 5;
                var y = 6;
                var sum = x + y;
                
                document.getElementById("demo4f").innerHTML = "The sum is " + sum;
                </script>
