Example Code

                let text10c = "";

JavaScript break

The break Statement in a Loop

Break the loop when the loop counter is 3:

Example Code


                                                    
                <script>
    
                let text10e = "";

                for (let i = 0; i < 10; i++) {
                    if (i === 3) { 
                        break; 
                    }
                    text10e += "The number is " + i + "<br> ";
                }

                document.getElementById("demo10e").innerHTML = text10e;            
                    
                </script>

JavaScript Loops

The continue Statement

Skip the iteration step when i equals 3.

Example Code


                                                            
                <script>
    
                let text10f = "";

                for (let i = 1; i < 10; i++) {
                    if (i === 3) { 
                        continue; 
                    }

                    text10f += i * 10 + "<br> ";
                }

                document.getElementById("demo10f").innerHTML = text10f;
        
                        
                </script>

JavaScript Loops

The for in Loop

A for in statement loops through the properties of a person object:

Example Code


                                                                    
                <script>
    
                const person = { fname: "John", lname: "Doe", age: 25 };
                
                let txt = "";
                for (let x in person) {
                    txt += person[x] + " ";
                }
                
                document.getElementById("demo10g").innerHTML = txt;   
                        
                </script>
