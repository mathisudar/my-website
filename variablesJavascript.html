JavaScript Variables

In this example, x, y, and z are variables.

Example Code


                    
                <script>
                var x = 5;
                var y = 6;
                var z = x + y;
                
                document.getElementById("demo5a").innerHTML = "The value of z is " + z;
                </script>

JavaScript Variables

In this example, price1, price2, and total are variables.

Example Code


                            
                <script>
                const price1 = 5;
                const price2 = 6;
                let total = price1 + price2;
                
                document.getElementById("demo5b").innerHTML = "The total is: " + total;
                </script>
